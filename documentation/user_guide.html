let operator

let function allows you to set some simulation variables by putting into a console a text like: let(, ) for exmaple: let("speed", 2) will set the speed of simulation factor to 2. NOTE: After changing a value of variable during running simulation, you shall refresh simulation to see an effect! There are some predefined variables_names (if you set one of them, then the simulation engine work will be changed accordingly to the description bellow). Nonetheless you are not limited to use only theese variables - you can define your own (by simply calling "let") and then use this vairable by calling simulation. in your script.

Predefined variables_names

let("gravity", False) let("earth_gravity", False) let("step_size", 0.02) let("speed", 2) let("direction", direction) "backward" or "forward" let("axis", True) let("grid", True)
